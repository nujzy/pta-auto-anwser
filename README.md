# PTA 智能答题助手 (PTA Intelligent Exam Assistant)

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/Version-1.0.1-blueviolet.svg)](#)

这是一个专为 PTA (拼题A) 平台研发的**全自动、智能化、自适应**答题插件。它基于原生 JavaScript 编写，完美绕过沙盒环境并无缝对接主流 React 底层框架，支持各种大语言模型，帮助你高效、自动化地完成 PTA 平台上的各类题型。

> ⚠️ **声明**：本脚本仅用于学术研究与个人习惯的定制化学习交流，请勿用于任何正式考试或违规作弊行为！

## 🛠️ 相比原版的修改与核心修复 (Modifications & Bug Fixes)

本项目在原作者优秀的骨架基础上，针对连续自动运行的场景，对控制流和状态同步逻辑进行了重构，主要解决以下实际运行中的问题：

1. **修复非法选择器导致的原地死循环 Bug**
   - **原版问题**：原代码在查找提交与下一题按钮时，错误地在原生的 `document.querySelector()` 中使用了 jQuery 专属的伪类选择器（如 `:contains("提交")`）。这导致底层直接抛出 `DOMException` 异常，被外层 `catch` 捕获后中断了后续流程，致使脚本无限卡在“分析->生成->报错重试”的死循环中。
   - **重构方案**：全面移除了非法的 `:contains()` 选择器，改用纯原生 JavaScript 遍历 DOM 节点并配合 `textContent.includes()` 进行精准的文本匹配。

2. **优化框架层状态同步 (针对新版 CodeMirror 6)**
   - **原版问题**：直接对输入框或受控组件赋值，在面对 PTA 最新采用的 CodeMirror 6 编辑器时，容易在失焦或鼠标点击时被 React 判定为篡改并导致代码清空。
   - **重构方案**：加入 **React Fiber 树遍历逻辑**。通过向上递归检索 DOM 节点上的 `__reactFiber$` 私有属性，直接调用组件底层的 `onChange` 钩子，确保代码填入与 React 内部 State 状态同步；配合原生 `setter` 绕过属性劫持。

4. **提高语言识别精度**
   - **原版问题**：依赖页面文本模糊匹配（如包含 `cpp` 即判断为 C++），在部分特殊题型下易发生误判。
   - **重构方案**：定位 PTA 核心下拉单选组件 `.select__single-value` 的真实选中值，提取当前编辑器切换的真实编程语言，提升 Prompt 上下文的准确率。

## ✨ 核心特性

* **🤖 广泛的模型兼容（OpenAI 风格 API）**
  * 内置支持 **硅基流动、DeepSeek 官方、OpenAI、阿里云百炼、智谱AI** 等主流大模型服务。
  * 支持一键动态获取并自动刷新服务商提供的模型列表。
* **🚀 深度重构的自动化控制流（拒绝原地死循环）**
  * 将“异步提交”与“切题跳转”彻底解耦。引入**智能评测状态轮询机制**，精准捕捉服务器并发评测结束的瞬间，彻底告别旧版因网络延迟卡在同一道题目重复作答的致命 Bug。
  * 在异常捕获块（Catch）中集成**逃离兜底机制**，即使单题突发未知错误也能强制跳过，保障无人值守时全自动连续答题的稳定性。
* **🧠 破防框架层：React Fiber 状态无缝同步**
  * 面对新版 PTA 的复杂编辑组件，引入 **React Fiber 虚拟 DOM 树深度遍历技术**。向上递归检索底层的 `__reactFiber$` 私有节点，直接调用核心组件的 `onChange` 状态钩子，实现代码填入与框架内部 State 的绝对同步。
  * 结合 `execCommand('insertText')` 物理级打字仿真与原生原型链 `setter` 调用，完美绕过 React 对受控组件的属性劫持。
* **🛡️ 独创沙盒拦截：物理粉碎切题弹窗**
  * 处于真实页面的上下文，利用最高优先级拦截并重写 `beforeunload` 注册事件，将 `window.onbeforeunload` 强制锁定为 `null`。
  * **彻底根治连续切题时浏览器频繁弹出“内容未保存”系统阻拦弹窗的痛点**，全自动流程丝滑不中断。
* **🧩 题型与语言精准自适应**
  * 目前自动答题完美支持 **函数题和编程题**。
  * **语言精准提取**：优先定位 PTA 核心下拉单选组件，精准捕获当前编辑器所选的真实编程语言（Python3、Java、C++、C 等）并自动进行去噪裁剪，确保喂给大模型的上下文提示词绝对精准。

## 🛠️ 安装与使用指引

### 1. 准备工作
确保你的浏览器已安装用户脚本管理器扩展（如 **Tampermonkey** / **Violentmonkey**）。

### 2. 一键安装
点击下方链接，油猴插件将自动捕获并提示安装最新版本（由于使用了标准 `.user.js` 后缀，点击后将直接唤醒油猴安装面板）：
👉 [**点击此处安装最新版脚本**](https://raw.githubusercontent.com/nujzy/pta-auto-anwser/main/auto-anwser.user.js)

### 3. 使用步骤
1. 打开进入任意 [PTA 题目页面](https://pintia.cn/)。
2. 按下 `Ctrl + Shift + A` 唤醒助手面板。
3. 在 **⚙️ 配置** 选项卡中，选择你的服务商（或手动填写 Base URL），输入你的 `API Key`。
4. 点击 **🔄 获取模型列表**，并在下拉框中选择你偏好的大模型。
5. 点击底部的 **💾 保存配置**。
6. 切换到 **📝 答题** 面板，点击 **▶️ 开始自动**，即可静观脚本进行一键分析、答题、填入、提交并自动跳转下一题的全自动化闭环流程。
